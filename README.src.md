## How to build & test

First, you need clone this repository locally:
```bash
git clone git@github.com:coinspec/data.git coinspec-data
cd coinspec-data
npm install
```

then you can use these commands:
```bash
# to generate bundle
npm run-script build

# to run tests
npm test 
```
